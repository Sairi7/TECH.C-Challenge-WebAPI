# 概要
## 使うAPI
* [グルーポンAPI (http://www.groupon.jp/about-api)](http://www.groupon.jp/about-api)

## 内容
* 位置情報などのエリア情報を取得して近隣の店舗のお得クーポンを取得する。

## 目標・ゴール
* 位置情報で緯度経度が使えるので、AndroidのGPS機能を使い、緯度経度を取得し、APIに渡してクーポンを取得する。

## その他
* できる人は、GoogleMapも使おうぜらしい。
